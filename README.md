# TarkovGeoMonitor
**TarkovGeoMonitor** は、Escape from Tarkov (EFT) のログファイルを監視し、接続しているレイドサーバーのIPアドレスとその物理的な所在地（国名）を表示するツールです。

時間帯によって相性の悪いサーバがあったり、パーティプレイ中にリーダーの回線と相性が悪くコネロスした際に、毎回ログファイルを開いて調べるのが面倒だったので自分用に作成していたものです。

![ScreenShot](https://github.com/user-attachments/assets/dfaba4fb-ed41-4879-8af7-8c3cf00dccee)

## 使い方
1. **ダウンロード**
   [Releases](https://github.com/kurua/TarkovGeoMonitor/releases) から zipファイルをダウンロードして解凍してください。

2. **起動**
   `TarkovGeoMonitor.exe` を実行するだけで、自動的にログを監視します。
   
   ※ IP判定用のデータファイル（.mmdb）は同梱されています。

### データの更新について（任意）
あまり効果は無いと思いますが、最新のデータを使いたい場合は [DB-IP (Lite)](https://db-ip.com/db/lite.php) から `.mmdb` ファイル（`dbip-country-lite-xxxx-xx.mmdb`）をダウンロードし、exeと同じフォルダに上書き保存してください。

## 注意事項
* ゲームクライアントがローカルに出力するテキストログファイルを参照するだけの仕組みであり、ゲームプロセスのメモリ参照やパケットの解析は行っていません。クライアントが出力したテキストログを読んでいるだけなので安全だとは思いますが、**使用は自己責任でお願いします。**
* 表示される国は DB-IP の情報がベースとなっており、EFTのサーバ選択画面の名称と一致するとは限りません。その辺はフィーリングで対応してください。
* あくまで自己満足で作ったプログラムなので、お使いの環境によっては動かないかもしれません。また、Steam版での動作確認は行っていません。

# TarkovGeoMonitor
**TarkovGeoMonitor** is a tool that monitors *Escape from Tarkov* (EFT) log files to display the IP address and physical location (country) of the connected raid server in real-time.

I developed this tool for personal use because I found it tedious to manually open and check log files every time I encountered connection issues, such as finding servers incompatible with my connection during certain times of the day or diagnosing connection losses when playing in a party.

## How to Use

1.  **Download**
    Download the zip file from [Releases](https://github.com/kurua/TarkovGeoMonitor/releases) and extract it.

2.  **Launch**
    Simply run `TarkovGeoMonitor.exe`. It will automatically start monitoring the logs.

    *Note: The IP geolocation data file (.mmdb) is included in the package.*

### Updating the Data (Optional)
While not strictly necessary, if you wish to use the latest IP data, you can download the `.mmdb` file (`dbip-country-lite-xxxx-xx.mmdb`) from [DB-IP (Lite)](https://db-ip.com/db/lite.php) and overwrite the existing file in the same folder as the executable.

## Important Notes

* This tool simply references the local text log files generated by the game client. It does **not** read game process memory or perform packet analysis. Since it only reads text logs output by the client, it is considered safe, but **please use it at your own risk.**
* The displayed country is based on DB-IP data and may not exactly match the server names shown in the EFT launcher. Please use your best judgment when interpreting the results.
* As this is a program created for personal use, it may not work in all environments. Additionally, operation with the Steam version has not been verified.
